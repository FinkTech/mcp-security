# START HERE (English)

This guide helps you adopt **mcp-security** quickly.

## What this repo provides

- 12 security rules (SEC-001 to SEC-012)
- Vulnerable vs secure code examples
- OWASP/CWE and compliance mappings (SOC2, HIPAA, PCI DSS)
- Bilingual docs (EN/ES)
- Auto-generated PDFs

## 5-minute path

1) Open the rule index: `../INDEX.md`  
2) Pick the rules you need (by severity and use case).  
3) Apply the secure patterns in your MCP server code.  
4) Update both languages if you contribute documentation changes.  
5) Regenerate PDFs if needed (see the scripts/workflow used by the repo).

# Security (English)

This page is an English entry point for security reporting and policies.

## Reporting vulnerabilities

Please follow the repository security policy:

- `../../SECURITY.md`

If you believe you found a vulnerability in:
- the guidance in a rule,
- the secure code examples,
- the PDF generation pipeline,
- or any scripts,

please report it via the process described in `../../SECURITY.md`.

## Notes

- Do not publicly disclose sensitive exploit details in Issues.
- Use responsible disclosure.
